# Zelda Online

Play at: https://jpdieffe.github.io/zeldaonline/

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Shift | Sprint |
| Space | Jump |
| Q | Roll |
| Left-click | Sword attack (3-hit combo) |
| Right-click | Defend |
| Scroll | Zoom camera |
| Click canvas | Lock mouse / aim camera |
| Esc | Release mouse |

## Multiplayer

- **Host Game** — creates a room code to share with a friend
- **Join** — enter a room code to connect P2P
- **Solo Play** — play offline

## Stack

- [BabylonJS 9](https://www.babylonjs.com/) — 3D engine
- [PeerJS](https://peerjs.com/) — P2P networking
- [Vite](https://vitejs.dev/) — build tool
- TypeScript
